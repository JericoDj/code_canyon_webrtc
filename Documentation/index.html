1) Add Firebase to your Flutter app

Go to Firebase and sign in with your Google account. https://console.firebase.google.com

2) File structure

Copy the "google-services.json" file you downloaded into the "lib/android/app/" folder.

Copy the "GoogleService-Info.plist" file you downloaded into the "lib/ios/" folder.

3) Edit files in "lib"

Open "android/app/google-services.json"

Update the information in the `lib/utils/firebase_options.dart` file with your own Firebase details.
